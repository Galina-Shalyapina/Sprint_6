# Автотесты для веб-приложения "Яндекс Самокат"

## Описание

Этот репозиторий содержит автотесты для веб-приложения "Яндекс Самокат". Тесты написаны на языке программирования Python с использованием фреймворка Selenium для автоматизации браузера.

## Тестовые сценарии

### Выпадающий список в разделе «Вопросы о важном»

Проверка:
- При нажатии на стрелочку открывается соответствующий текст.
- Отдельный тест для каждого вопроса.

### Заказ самоката

Проверка позитивного сценария с двумя наборами данных:
1. Нажать кнопку «Заказать» (две кнопки: вверху и внизу страницы).
2. Заполнить форму заказа.
3. Проверить появление всплывающего окна с сообщением об успешном создании заказа.
4. Проверить переход на главную страницу «Самоката» при нажатии на логотип «Самоката».
5. Проверить переход на главную страницу Дзена при нажатии на логотип Яндекса.

## Требования

- Python 3.10
- Selenium
- Allure

## Установка

1. Клонируйте репозиторий:
    ```sh
    git clone https://github.com/Galina-Shalyapina/Sprint_6.git
    ```
2. Установите зависимости:
    ```sh
    pip install -r requirements.txt
    ```

## Запуск тестов

Для запуска тестов используйте команду:
```pytest```

## Генерация отчета Allure
Для просмотра отчета необходимо установить фреймворк Allure
```allure serve allure_results```
